Running App on Device
After building the app, what next? Well, before we can run the app in an emulator or a real device, we'll need to download and configure couple of things.

First, ensure that you have Java Development Kit 8 or later

Install Android Studio or Standalone SDK Manager. Detailed installation instructions are on Android's developer site.
You must also install the packages for whatever API level you wish to target. It is recommended that you install the highest SDK version. To do that, open the SDK Manager and install the following

  • Android Platform SDK for your targeted version of Android
  • Android SDK build-tools version 19.1.0 or higher
  • Android Support Repository (found under "Extras")
