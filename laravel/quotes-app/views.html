Favorite Quotes App: Views

Objectives: We'll build an app to keep track of list of our favorite quotes. It's going to be very basic. However, We'll learn the following

  • Installing laravel
  • How laravel MVC works
  • Performing CRUD (Create, Read, Update, Delete) operations with MySQL backend

Installing Laravel
We'll start by installing laravel and creating our project. There are couple of options available but I think the easiest option in windows is to

  • Download and install Xampp. Head over to Apache Friends and download the latest version
  • Download and install composer. Composer is a dependency manager for PHP. visit download page
  • Open a command terminal. Change directory to the htdocs of xampp and run the command to create a new laravel project

  • Launch Notepad as an administrator. Edit http-vhosts.conf file found in C:/xampp/apache/conf/extra. Add

  • Still using notepad, Edit host file found in C:\Windows\System32\drivers\etc. Add

  • Run Xampp and start the apache and mysql modules. Open a browser and enter quotes.dev You should see the welcome page below (or something similiar)

If you encountered errors. Remember we have a savior in Google. Search for the solution!
After creating project successfully, open project in your favorite code editor (Maybe Sublime Text, VS Code, Atom or others ).

Laravel Folder Structure
Laravel follows a MVC structure. The view, model and controllers are separated.

We are not going to discuss all the functions of the folders and files for now, that can be very boring. However, the app/Http/Controllers folder is where we define the controllers.

We can place our images, styles and scripts in the public folder. The resources/views folder contains our views.

The web.php file in the routes folder is where we defined the routes.

Creating our first view
Imagine that we are building a site with so many pages, we can use a controller to manage the pages. So, before we add a page, let us add a controller first.

We can generate a controller using a command line. We'll use the artisan command. Artisan is Laravel command line tool that helps us to perform tasks that we hate to do manually. Using Artisan, we can create models, views, controllers, migrations and many other things.
